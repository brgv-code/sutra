# Using Next.js with TypeScript

Next.js is a powerful React framework that enables you to build server-side
rendered and statically generated web applications. When combined with
TypeScript, it provides an even more robust development experience. Let's
explore how to set up and use Next.js with TypeScript.

# Setting up a Next.js project with TypeScript

To create a new Next.js project with TypeScript, you can use the following
command:

```bash
npx create-next-app@latest --typescript

```
